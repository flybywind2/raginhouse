📄 Import from Confluence DC

You can find the Page ID in the URL when viewing a Confluence page

ℹ️ How to find Page ID

  1. Go to the Confluence page you want to import
  2. Look at the URL in your browser
  3. Find the number after "/pages/" or "/display/" - this is your Page ID
  4. Example: https://confluence.company.com/display/SPACE/Page+Title?pageId=123456789
